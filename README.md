# TableView
TableView support many features
* Cell style
* Border line style
* Font style
* Rich text
* Object in cell
* Support click on object/text/cell
* Support cell merge
* Support scroll
* Support row and column headers
* Support resize row height and column width
* Support range selection
* ......
